2026-05-11
Speculation Rules + View Transitions + Course schema Feature

Adopted three 2024-2026 web-platform APIs:

  • Speculation Rules API (Chrome 121+) — prerenders adjacent pages on hover. Inter-page nav becomes instant.
  • View Transitions API (Chrome 126+, Safari 18.2) — smooth fade between pages.
  • schema.org/Course — six courses eligible for Google's course-listing rich result.
  • Speakable schema on insights post — voice assistants can read highlights.

2026-05-11
Algorithm Lab: search, keyboard shortcuts, step mode, persistence Feature

Power-user upgrades to the Algorithm Lab:

  • Search filter (⌘K) — by name, creator, category
  • Keyboard shortcuts: Space, R, →, ↑↓, C, +/−, ? (help overlay)

2026-05-08
Algorithm Lab: 17 new algorithms Feature

Algorithm Lab now covers 39 algorithms across 7 categories spanning 2,300+ years (Euclid → Tim Sort). New categories: Math (Sieve, Euclid GCD, Pascal), DP (Fibonacci, Kadane, LCS, LIS, Edit Distance, Knapsack), Strings (KMP, Rabin-Karp), plus Bellman-Ford, Kruskal, and four more sorts (Cocktail, Comb, Cycle, Tim).

2026-05-07
3-column Algorithm Lab + STATE inspector Feature

Restructured Algorithm Lab into permanent 3-column layout: sidebar · 3D scene · code panel with synchronised line highlighting. New STATE inspector below the code shows live variable values during execution.
