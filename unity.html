Drop your Unity 6 WebGL build into the repo and this space comes alive. Three steps to launch.

1
Export from Unity 6
File → Build Settings → WebGL → Build. Enable Compression Format: Disabled for easy static hosting.
2
Drop the build folder here
Copy the exported folder into website/unity-build/. It should contain index.html + Build/.
3
Push to GitHub
git add website/unity-build && git push — GitHub Pages serves it instantly. No config needed.

Once unity-build/index.html exists, this page auto-switches to the full Unity canvas with a loading bar and fullscreen button.
